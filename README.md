🍽️ Star Cuisine - Digital Menu Website
A premium, mobile-responsive digital menu designed for Star Cuisine, a local restaurant in Khurrianwala. This project focuses on high-end aesthetics (Dark Mode) and seamless user experience to bridge the gap between browsing and ordering.

🚀 Live Demo
Check out the live site: https://starcuisine.vercel.app/

✨ Features
Premium Glassmorphism UI: A modern dark-themed design with a "Royal" feel using custom CSS.

Mobile-First Approach: Fully responsive layout optimized for smartphone users.

WhatsApp Integration: Direct "Order via WhatsApp" buttons to increase customer conversion.

Categorized Menu: Organized sections for BBQ, Pizza, Burgers, and Family Thalis.

Fast Loading: Deployed on Vercel for 99.9% uptime and lightning-fast speeds.

🛠️ Tech Stack
HTML5: Semantic structure for better SEO.

CSS3: Custom variables, Flexbox/Grid, and animations.

Deployment: Vercel.

Version Control: Git & GitHub.
